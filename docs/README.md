## Contributing

When adding new documentation:

1. Create a new subdirectory for the topic if it doesn't exist (e.g., `analytics/`, `deployment/`, `frameworks/`)
2. Use descriptive filenames with hyphens (e.g., `getting-started-with-web-analytics.md`)
3. Include clear headings and sections
4. Add code examples with proper syntax highlighting
5. Update this README to include links to new documentation
